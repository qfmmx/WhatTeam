# AQI-SYS 技术文档

## 1. 项目概述

### 1.1 项目简介
AQI-SYS（空气质量指数系统）是一个基于 Spring Boot 开发的空气质量监测与管理系统，主要用于公众监督反馈、网格员管理、数据统计分析等功能。

### 1.2 项目名称
AQI-SYS

### 1.3 项目版本
0.0.1-SNAPSHOT

## 4. 核心功能模块

### 4.1 管理员认证模块
- 管理员登录
- JWT 身份验证
- 拦截器认证

### 4.2 公众监督反馈模块
- 分页查询反馈列表
- 获取反馈详情
- 查询本市在岗网格员
- 查询跨省可调派网格员
- 指派网格员
- 更新反馈状态

### 4.3 区域与网格员模块
- 查询所有省份
- 根据省份查询城市列表
- 根据城市查询网格员列表

### 4.4 统计分析模块
- 分页查询统计记录
- 获取单条统计记录详情
- 各省超标数据统计
- AQI 等级分布统计
- 超标趋势统计（近12个月）
- 概览数据指标（累计检测总数、空气良好天数、省份覆盖率、城市覆盖率）

## 5. 核心类说明

### 5.1 统一返回结果类 (Result.java)
- **路径**: `com.it.aqisys.result.Result`
- **说明**: 全局统一响应包装类
- **主要方法**:
  - `ok(T data)`: 成功响应
  - `fail()`: 失败响应
  - `build(T body, ResultCodeEnum)`: 构建响应

### 5.2 JWT 工具类 (JwtUtil.java)
- **路径**: `com.it.aqisys.util.JwtUtil`
- **说明**: JWT 令牌生成与解析
- **主要方法**:
  - `createToken(Long userId, String username)`: 生成 Token（有效期1小时）
  - `parseToken(String token)`: 解析 Token

### 5.3 AQI 反馈实体类 (AQIFeedBack.java)
- **路径**: `com.it.aqisys.pojo.AQIFeedBack`
- **说明**: 空气质量公众监督反馈信息
- **主要字段**:
  - `afId`: 反馈编号
  - `telId`: 公众监督员手机号
  - `provinceId`: 省区域编号
  - `cityId`: 市区域编号
  - `state`: 状态（0:未指派 1:已指派 2:已确认）

## 10. 项目特点

1. **分层架构**: 遵循 MVC 分层设计，职责清晰
2. **MyBatis-Plus**: 使用增强版 ORM 框架，提高开发效率
3. **统一响应**: 全局统一返回结果封装
4. **JWT 认证**: 基于 Token 的身份验证
5. **Druid 监控**: 集成 Druid 连接池，提供 SQL 监控功能
6. **全局异常处理**: 统一的异常处理机制
